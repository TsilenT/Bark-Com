Strategic-layer rework that ties together:
1. **Threat scaling:** easy missions push threat up faster; some mission rewards reduce threat.
2. **Threat-resets:** at 100% threat, a base-invasion mission fires and resets to 0.
3. **Doom counter:** tracks how many resets have happened; color-coded levels.
4. **Final fight:** on the 3rd reset, the run climaxes against Dogthulu (win/lose run end).
5. **Mission-slot upgrades:** start with 2 visible missions, upgrade to 3 then 4 for kibble (~250/500).

Note: a Dogthulhu boss moveset already shipped — fold that into the finale design.
