# Per-biome disruptive elements affecting xp/rewards

**Category:** Feature
**Source:** TODO.txt (0.6 extra features, also referenced in 1.0)
**Original note:** "add disruptive elements to mission (has effect on xp rewards etc.) based on biome. Disruptive element displayed on mission screen as well"

Each biome can roll a "disruptive element" (e.g., low visibility, acid rain) that modifies xp/rewards. Surface the active disruption on the mission-select card.
